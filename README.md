# Survey Swap

A full-stack web application for sharing and participating in surveys.

## Tech Stack

### Frontend
- React + Vite
- Modern UI components

### Backend
- Node.js + Express
- RESTful API

## Project Structure

```
Survey Swap/
├── frontend/          # React frontend
├── backend/           # Express backend
└── README.md
```

## Getting Started

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on: http://localhost:5000

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on: http://localhost:5173

## API Endpoints

- `GET /` - Welcome message
- `GET /api/surveys` - Get all surveys

## Development

Both frontend and backend need to run simultaneously:
1. Open terminal 1: Run backend (`cd backend && npm run dev`)
2. Open terminal 2: Run frontend (`cd frontend && npm run dev`)
